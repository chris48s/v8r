# Introduction

v8r is a command-line validator that uses [Schema Store](https://www.schemastore.org/) to detect a suitable schema for your input files based on the filename.

## Getting Started

One-off:

```bash
npx v8r@latest <filename>
```

Local install:

```bash
npm install -g v8r
v8r <filename>
```
